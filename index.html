Current projects

Lime2D

An open source in-game map/animation editor for 2D C++ SFML game development projects. It is designed to be an easy library to set up and use in your game. Since it is an in-game editor, you can edit maps in real-time with your player, physics, collisions, and everything else being rendered. This will make for extremely efficient 2D level creation.
